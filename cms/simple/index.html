Вы попали на index.html.

Это страница назначения при перенаправлении файлов с расширением xmx.

  • Показать (show.html)
  • Stop - заблокированная страница
  • Redirect - редирект на show.html
  • Подкаталог

Тесты mod_rewrite для nginx:

  1. Запросите .xmx файл → перенаправление на index.html
  2. Запросите stop.html → 403 Forbidden (доступ запрещен)
  3. Запросите redirect.html → покажет show.html
